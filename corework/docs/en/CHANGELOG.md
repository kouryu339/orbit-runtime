# Changelog

This English changelog summarizes the major documentation and architecture
milestones for `corework`.

## Highlights

- Added scoped cache and runtime context primitives.
- Added state machine and event abstractions.
- Added Blueprint workflow nodes and execution model.
- Added field-path access helpers and tests.
- Added RPC Tool protocol v1 for sidecar-based tools.
- Added workflow draft, recorder, executor, snapshot, and command modules.
