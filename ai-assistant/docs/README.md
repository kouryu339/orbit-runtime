# ai-assistant System Documentation

This directory explains the internal AI system: conversation state, Agent
coordination, tool execution, Skills, prompts, persistence, and runtime
mechanics.

It is not the public integration guide and not the FFI contract reference.

| Documentation area | Responsibility |
|---|---|
| `ai-assistant/docs` | How the AI system works internally. |
| `examples/guides` | How product teams integrate the framework. |
| `sdk` | Runtime Host SDK and Agent Tool SDK usage. |
| `agent_runtime_ffi/docs` | Stable ABI, event, command, and schema contracts. |
